安装
==============

### CocoaPods

1. 在 Podfile 中添加 `pod 'DBJSONModel'`。
2. 执行 `pod install` 或 `pod update`。
3. 导入 \<DBJSONModel/DBModelHeader.h\>。

### 手动安装

1. 下载 DBJSONModel 文件夹内的所有内容。
2. 将 DBJSONModel 内的源文件添加(拖放)到你的工程。
3. 导入 `DBModelHeader.h`。


系统要求
==============
该项目最低支持 `iOS 6.0` 和 `Xcode 8.0`。
